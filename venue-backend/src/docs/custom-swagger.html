Atomic Seat Reservation
// Single atomic op â€” check + decrement
const slot = await Slot.findOneAndUpdate(
  { _id: slotId, availableSeats: { $gte: quantity } },
  { $inc: { availableSeats: -quantity } },
  { new: true, session }
);

No Overselling

The $gte + $inc execute atomically. MongoDB serializes concurrent ops â€” 100 users, 10 seats, 0 oversells.
